# Oracle SQL & PL/SQL Practice

This repository contains Oracle SQL and PL/SQL practice scripts for learning database concepts.

## 📁 Files

- `basic.sql` → Employees table operations, joins, subqueries, window functions, PL/SQL blocks, procedures, triggers.
- `oracle_customer_dml_transactions.sql` → Customer table operations, DML (INSERT, UPDATE, DELETE), transaction control (COMMIT, ROLLBACK, SAVEPOINT), and constraints.

## 🧠 Topics Covered

- SQL DDL & DML
- Aggregations & Joins
- Subqueries & Window Functions
- PL/SQL (loops, cursors, conditions)
- Procedures & Triggers
- Transaction Control (TCL)
- Date & String functions

## 🚀 Usage

Run the scripts in any Oracle SQL environment (SQL Developer / TOAD) to practice and learn concepts.

## 🎯 Purpose

For Oracle SQL and PL/SQL learning, practice, and interview preparation.
